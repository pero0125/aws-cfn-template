【注意事項】

このフォルダには、
　１）ハンズオンで AWS 上にアップロードするファイル
　２）各動画で修正したあとのファイル（上手く動かないときにコピペしていただくなどの利用を想定）
の2種類のファイルがあります。


１）のファイルに該当するものについては、★マークをつけていますので、そのまま S3 や Cloud9 環境にアップロードしてお使いください。
また、【動画N mm:ssごろ】のように、ハンズオン動画の中で使用するタイミングについても明記しています。

２）は参考資料になります。必要があれば答え合わせとしてお使いください。

また、そのまま AWS にアップロードするファイル *以外* については、拡張子を ".txt" としています。（例：index.html.txt）
.html や .json などのファイルを開く環境（エディタ）がない方でも、ハンズオンを進めていただけるようにこのようにしています。
